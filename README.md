# T-Code Solutions

## "El puente entre el lenguaje humano y el binario."

---

### 📚 Sobre el repositorio

Este repositorio ha sido creado para alojar y gestionar todas las asignaciones, proyectos y material práctico correspondiente a la materia universitaria **Lenguajes y Compiladores**.  
Cada entrega refleja el trabajo colaborativo del grupo, aplicando los conceptos teóricos sobre análisis léxico, sintáctico, semántico y generación de código.
